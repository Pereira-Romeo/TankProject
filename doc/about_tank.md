It uses **[pygame](https://www.pygame.org/wiki/about)** to handle controllers.

## Controls

The tank can be controlled by the following controllers:
- **ps3**  (USB)
- **ps4**  (bluetooth)
- **ps5**  (bluetooth)
- **Xbox** (bluetooth)

<ins>Tank movement: </ins> \
R2 / RT  = right track forward \
R1 / RB  = right track backward \
L2 / LT  = left track forward \
L1 / LB  = left track backward

<ins>Turret base movement: </ins> \
Arrow right = rotate right \
Arrow left = rotate left
